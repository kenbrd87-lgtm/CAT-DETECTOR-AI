# Cat Detector AI
Building AI course project

## Summary
My project is an AI that looks at photos and tells if there is a cat or not. It uses simple image recognition like we learned in the Elements of AI course. This helps people find their cat photos faster.

## Background
**Which problems does it solve?**
* People have many photos and searching for cat photos takes time
* My motivation: I love cats and I want to learn AI

**Why is this topic important?**
AI saves time and helps us organize photos automatically instead of searching manually.

## How is it used?
1. User uploads a photo to the app
2. AI checks the photo using filters for cat ears and eyes
3. AI says "Cat found!" or "No cat here"
4. Works best with clear daylight photos

## Data sources and AI methods
**Where does data come from?**
I will use free cat photos from Kaggle.com "Cats vs Dogs" dataset.

**AI method:**
Convolutional Neural Network CNN with filters, like the exercise we did in the course.

## Challenges
**What does my project NOT solve?**
* It cannot tell the cat's name or breed
* It does not work with very dark or blurry photos

**Ethical considerations:**
I will only use photos that people share publicly. No private photos without permission.
